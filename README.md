# TagPilot - Customer DNA Analysis

TagPilot 顧客標籤行銷引擎 - 客戶DNA分析與行銷決策支援系統

## Deployment

Deploy on Posit Connect Cloud from this GitHub repository.

### Required Environment Variables

| Variable | Description |
|----------|-------------|
| `SUPABASE_DB_HOST` | Supabase database host |
| `SUPABASE_DB_PORT` | Database port (default: 6543) |
| `SUPABASE_DB_USER` | Database user |
| `SUPABASE_DB_PASSWORD` | Database password |
| `SUPABASE_DB_NAME` | Database name (default: postgres) |
| `OPENAI_API_KEY` | OpenAI API key for AI features |
